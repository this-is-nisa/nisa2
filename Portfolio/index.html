SAT Timer

A proctor in the SAT alerts you when you have half the time remaining and when you have five minutes left. I couldn't find any timer that had this function, so I decided to make my own.

This is a Javascript timer tool that functions as a normal timer, but has an announcement when you have half-time or 5 minutes remaining. I made it to mimic the alerts you get from the proctor when taking the real SAT. You can choose which functions you would like and includes preset times for each test. There are still a few minor bugs, but I will get them fixed soon. I have used it for my own practice tests, and I hope it can be useful for SAT studiers in the future.
